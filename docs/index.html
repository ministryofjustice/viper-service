VIPER Rating

Retrieve VIPER Rating

GET /viper/{nomsId}

Retrieves a single VIPER Rating from the dataset Pre-seeded values for smoke testing: B8888BB, I1111II, O0000OO

nomsId

A known NOMS ID

type
string
in
path
200 OK
viperRatingResult

Success

400 Bad Request
invalidArgumentError

Invalid NOMS Id

404 Not Found
notFoundError

Not Found

Response Example (200 OK) 
{
  "nomsId": "I9876RA",
  "viperRating": 0.56
}
Response Example (400 Bad Request) 
{
  "code": "Invalid Argument",
  "message": "Request validation failed: Parameter (nomsId) does not match required pattern: [A-Z]\\d{4}[A-Z]{2}"
}

{
  "code": "Not Found",
  "message": "Resource Not Found"
}
